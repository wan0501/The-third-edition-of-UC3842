# UC3842 第三版 — 85VAC~265VAC 输入 / 24VDC 3A 输出反激开关电源
本项目是一个基于 **UC3842**  PWM 控制芯片的**反激（Flyback）式开关电源**完整设计，提供 **85 VAC ~ 265 VAC 宽范围交流输入**、**24 VDC / 3 A（72 W）稳压直流输出**。
这是该电源的**第三版**设计，在之前版本基础上对电路参数、变压器设计和布局进行了全面优化。

## 项目概述

本项目以 UC3842 电流模式 PWM 控制器为核心，设计一款适用于工业控制、仪器仪表、辅助供电等场景的隔离型反激开关电源。第三版重点对以下方面进行了改进：

- **优化变压器**：重新计算电感量、匝比和磁芯选择，提高效率
- **改进 EMI 滤波**：增强输入滤波和共模抑制
- **完善保护功能**：过流保护（OCP）、过压保护（OVP）、欠压锁定（UVLO）
- **优化布局**：减少噪声耦合，改善热管理

### 应用场景

- 工业控制系统辅助电源
- 仪器仪表供电
- 白色家电控制板电源
- 通用 AC-DC 适配器 / 基板电源

## 电路架构

### 系统框图

```
AC输入 (85~265V) → EMI滤波 → 桥式整流 → 输入滤波电容 → 反激变压器 → 输出整流滤波 → 24VDC 输出
                           ↑                        ↓
                       PWM 驱动 (UC3842) ←─── 反馈环路 (TL431 + 光耦)
```

### 主要功能模块

#### 1. 输入级
- **保险丝**：输入过流保护
- **NTC 热敏电阻**：抑制启动浪涌电流
- **共模扼流圈 + X 电容**：EMI 共模滤波
- **桥式整流器**：全波整流
- **大容量电解电容**：整流滤波，提供平滑直流母线

#### 2. 控制级 — UC3842
- **启动电阻**：上电启动，启动后由辅助绕组供电
- **RC 振荡**：设定开关频率（约 65 kHz）
- **电流检测电阻**：逐周期峰值电流限制
- **软启动**：缓启动电路，避免启动过冲

#### 3. 功率级 — 反激变换
- **MOSFET 开关管**：高压功率管
- **RCD 吸收回路**：抑制漏感尖峰电压
- **反激变压器**：储能、隔离、电压变换

#### 4. 输出级
- **肖特基整流二极管**：高效输出整流
- **LC 滤波**：降低输出纹波
- **假负载电阻**：保障空载稳定性

#### 5. 反馈环路
- **TL431 精密基准**：输出采样与误差放大
- **PC817 光耦**：隔离反馈，提供初级侧 PWM 控制

## 关键设计要点

### 变压器设计

变压器是反激电源的核心。计算表格 `85AC~265VAC-DC24V3A.xlsx` 中包含了详细的设计过程：

- **磁芯选型**：根据功率等级选择合适磁芯（如 EE / PQ / RM 系列）
- **匝比计算**：根据输入电压范围和输出电压确定最优匝比
- **电感量**：在 CCM / DCM 模式下优化电感量，平衡峰值电流和磁芯损耗
- **气隙计算**：防止磁饱和
- **绕组结构**：优化初级、次级、辅助绕组的排布，降低漏感
- **铜损 / 铁损**：校核温升

> **注意**：变压器参数已根据第三版优化调整，请勿直接套用前两版参数。

### RCD 吸收回路

- **LC 串联谐振频率计算工具**（`22LC串联谐振频率计算工具.xlsx`）辅助选择吸收电容和电阻
- 目标：将漏感尖峰电压控制在 MOSFET 耐压范围内（通常降额 80%~85%）
- 平衡：吸收效果 vs 损耗

### 反馈补偿

- Type II 补偿网络（TL431 + RC 网络）
- 穿越频率设为开关频率的 1/5 ~ 1/10
- 保证足够的相位裕度（>45°）

### 布局要点（建议）

- **功率回路**：最小化高频电流环路面积
- **控制地 / 功率地**：单点接地 / 星形接地
- **隔离间距**：初级 - 次级间确保安全爬电距离
- **散热**：MOSFET 和整流二极管留足铜皮散热

## 使用说明

### 查看原理图

1. **方式一（推荐）**：使用 Altium Designer 打开 `UC3842第三版.SchDoc`
2. **方式二**：直接打开 `UC3842第三版.pdf` 查看原理图 PDF

### 使用设计计算表格

- `85AC~265VAC-DC24V3A.xlsx`：修改黄色的输入参数单元格，自动计算变压器参数
- `22LC串联谐振频率计算工具.xlsx`：输入电感和电容值，自动计算谐振频率

### 注意事项

- **高压危险！** 调试时建议使用隔离变压器
- 初次上电建议串联调压器 + 灯泡限流
- 修改变压器参数后请重新验证磁通摆幅（ΔB），防止磁饱和
